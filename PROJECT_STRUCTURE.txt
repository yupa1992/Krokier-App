Icons:
  - Lucide React 0.294

Development:
  - ESLint 8.55
  - TypeScript Types (React)


DATA FLOW
=========

User Action → Component State → Map Update → Save/Export

1. Drag Symbol from Sidebar
   ↓
2. Drop on Map (DropZone)
   ↓
3. Add to symbols[] state
   ↓
4. Render as DraggableMarker
   ↓
5. User can move/label/delete
   ↓
6. Save as JSON or Export
